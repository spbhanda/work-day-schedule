## Project Title: Work Day Schedule

This is a simple calender application that allows a user to save events for hour of the day.

## Project Description:

The webpage was built using HTML, CSS and Javascript.
This is a simple calender application that allows a user to save events for hour of the day. This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery.
